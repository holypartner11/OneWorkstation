[English](./README_en.md) | 中文

**版本：v0.0.1**

> ⚠️ 目前处于 UI 原型阶段 - 请勿用于生产环境。

## 简介

OneWorkstation 是一个统一的 AI 工作站平台，整合多个 AI 工具和助手。

## 技术栈

- [React](https://react.dev/) + [TypeScript](https://www.typescriptlang.org/)
- [Vite](https://vitejs.dev/)
- [Node.js](https://nodejs.org/) (后端)

## 快速开始

```bash
npm install
npm run dev
```

## 项目结构

```
OneWorkstation
├── src/              # 前端源代码
├── server/           # 后端服务
├── public/           # 静态资源
└── dist/            # 构建输出
```
