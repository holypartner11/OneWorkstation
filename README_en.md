# OneWorkstation

[中文](./README_zh.md) | English

**Version: v0.0.1**

> ⚠️ Currently in UI prototype phase - not for production use.

## Overview

OneWorkstation is a unified AI workstation platform that integrates multiple AI tools and assistants.

## Tech Stack

- [React](https://react.dev/) + [TypeScript](https://www.typescriptlang.org/)
- [Vite](https://vitejs.dev/)
- [Node.js](https://nodejs.org/) (backend)

## Getting Started

```bash
npm install
npm run dev
```

## Project Structure

```
OneWorkstation
├── src/              # Frontend source code
├── server/           # Backend server
├── public/           # Static assets
└── dist/             # Build output
```
